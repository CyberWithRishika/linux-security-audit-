# Linux Security Audit Script

## Overview
A Bash script that performs basic Linux security checks.

## Features
- Current user detection
- System information gathering
- Open ports enumeration
- Sudo user detection
- Firewall status check
- Available updates check

## Technologies Used
- Linux
- Kali Linux
- Bash

## Usage

```bash
chmod +x audit.sh
./audit.sh
```
